# AXIOMA

Protocolo de governança da colaboração humano-IA. Bilíngue PT/ES — carregue **um** idioma por projeto.
Protocolo de gobernanza de la colaboración humano-IA. Bilingüe PT/ES — carga **un** idioma por proyecto.

Todo o kit está na pasta [`axioma/`](axioma/). Comece pelo [`axioma/README.md`](axioma/README.md) — ele traz o mapa completo, instruções de instalação (Claude Code / claude.ai) e o guia de "60 segundos".

- Núcleo: [`axioma/pt/AXIOMA.md`](axioma/pt/AXIOMA.md) · [`axioma/es/AXIOMA.md`](axioma/es/AXIOMA.md)
- Changelog: [`axioma/CHANGELOG.md`](axioma/CHANGELOG.md)
- Projeto de referência (N2): [`axioma/exemplo/`](axioma/exemplo/)
- Validador: [`axioma/tools/axioma-lint.py`](axioma/tools/axioma-lint.py) (Python 3.8+, stdlib apenas)

## Instalação rápida

```bash
# copie a pasta axioma/ para a raiz do seu repositório
cp -r axioma/ /caminho/do/seu/projeto/
```

Detalhes completos em [`axioma/README.md`](axioma/README.md).
